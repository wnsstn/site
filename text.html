Переменные

Переменными назначаются имена, в них записываются значения, и затем они используются в программе по мере необходимости.

МОЖНО использовать
  • латинские буквы (A-Z, a-z)
  • русские буквы (не рекомендуется!)
  • ﻿﻿цифры (НО имя не может начинаться с цифры)
  • знак подчеркивания _
НЕЛЬЗЯ использовать
  • скобки
  • ﻿﻿знаки +, =,!, ? и др.

«=» - оператор присваивания

Типы переменных:

  • int - целое - a=5
  • float - вещественное - a=4.5
  • bool - логическое выражение - True
  • str - символьная строка - "Python"

Математические операции

Существуют следующие математические операции:

  • x + y - сложение
  • x - y - вычитание
  • ﻿﻿x * y - умнжение
  • x / y - деление
  • x ** y - возведение в степень

Для целых чисел дополнительно доступны операциии:

  • x // y - целочисленное деление
  • x % y - остаток от деления

Условный оператор

Условный оператор используется в том случае, когда требуется, чтобы код выполнялся при определённых условиях.

Оператор if является началом условной конструкции.
Завершается условие символом «двоеточие».
Затем — обязательный отступ в четыре пробела.
По завершении тела может идти следующее условие, которое начинается с оператора elif.
Последним идёт оператор else, который не содержит условия, а выполняется, только если ни одно из предыдущих условий не выполнилось.

Знаки отношений:

  • >< - больше, меньше
  • >= - больше или равно
  • <= - меньше или равно
  • == - равно
  • != - не равно

Сложное условие

  • and - логическое «И». Условие истинно, если оба истина.
  • or - логическое «ИЛИ». Условие истинно, если хотя бы одного из них истина.
  • not - логическое «НЕ». Изменяет логическое значение на противоположное.
